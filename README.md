# AI-lessson-program1
人工智能导论项目的仓库创建，以贪吃蛇为主题
\# 🐍 Snake AI: 基于深度强化学习的贪吃蛇游戏智能体



本项目实现了两个版本的贪吃蛇AI：

1\.  \*\*DQN (Deep Q-Network) AI\*\*: 使用PyTorch构建神经网络，通过与环境交互自我学习。

2\.  \*\*Super AI (A\* Pathfinding)\*\*: 使用经典的A\*寻路算法，实现近乎完美的路径规划。



你可以选择训练一个AI，或者直接运行一个预设的、非常聪明的AI来观看它如何玩游戏！

\## 📦 功能特性



\-   \*\*两种AI模式\*\*:

&#x20;   -   \*\*学习型AI (`snake\_ai.py`)\*\*: 基于DQN算法，从零开始学习如何玩贪吃蛇。

&#x20;   -   \*\*规则型AI (`super\_snake\_ai.py`)\*\*: 基于A\*算法，实时计算最优路径，几乎不会撞墙或撞到自己。

\-   \*\*人类游玩模式 (`snake\_human.py`)\*\*: 你可以亲自上手体验游戏。

\-   \*\*模型管理\*\*: 支持保存、加载和继续训练模型。

\-   \*\*可视化训练过程\*\*: 实时绘制训练分数和平均分数曲线。

\-   \*\*预训练模型生成器\*\*: 快速生成一个带有模拟训练历史的“智能”初始模型。
